# MEGA-2D
MEms-based Generic Actuation for 2D materials

*Cao Reconfigurable Quantum Materials Lab, University of California at Berkeley*

### Build GDS Files
```sh
# clone this repo
$ git clone https://github.com/DanielHeEGG/mega_2d
$ cd mega_2d

# initialize local venv
$ python -m venv venv
$ source venv/bin/activate

# install dependencies
$ pip3 install -r requirements.txt

# build
$ python3 build.py
```
